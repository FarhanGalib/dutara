# Dutara

## Dutara is an e-commerce application where user can registration, sign in, see products, add to cart, checkout, update own profile and see order list with order date and order current status. Admin can add new users, products, product category and can edit and delete them. Also can see orders and can change order status.

## Tools

1. fakecommerce API ( https://github.com/arnab-shuvo/fakecommerce )
2. React
3. Redux
4. Redux Thunk
5. Material UI
6. React Router
7. VS Code

## Pre-requisites

1. Npm

## How To Run

1. At first clone the project.
2. Run `yarn` or `npm install` to install all the dependencies
3. Once installed, run `yarn start` or `npm start`
4. The project will run by default on `3000` port.

## Features

-   Customer

1.  Sign in/ sign up using `JWT` token.
2.  View products in home page.
3.  search products and sort product by category.
4.  pagination
5.  Add to cart.
6.  Proceed to checkout.
7.  Edit Profile.

-   admin

1.  Sign in/ sign up using `JWT` token.
2.  View products in home page.
3.  search products and sort product by category.
4.  pagination
5.  View orders and change Order status (delivered, pending, canceled)
6.  Add Products, edit and delete them
7.  Add Customer, edit profile and delete them
8.  Add Category of product, edit and delete them
